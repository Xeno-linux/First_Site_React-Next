# First_Site_React-Next

prérequis: 
NodeJS

Création du projet
1. commande terminal: npx create-next-app@latest
2. ok to proceed(y) ? répondre y
3. what is your project name ? répondre first_site
4. Would you like to use the recommended Next.js defaults? répondre avec No, customize settings
5. typescript: No
6. linter: None
7. react compiler: No
8. Tailwind CSS: No
9. src/ directory ? Yes
10. App Router ? Yes
11. Customize Import alias ? No



Démarrage du projet:
1. aller dans le dossier du projet
2. npm run dev
3. aller voir via un navigateur sur: http://localhost:3000/
